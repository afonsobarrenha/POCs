# terraform

## AWS

```
aws configure
```

## Comandos

```
terraform init
terraform apply
terraform show
chmod 400 lab-key.pem
ssh -i "lab-key.pem" ubuntu@ec2-1-2-3-4.compute-1.amazonaws.com
terraform destroy
scp -i "lab-key.pem" <diretorio-do-arquivo>/loja.war ubuntu@ec2-54-94-172-57.sa-east-1.compute.amazonaws.com:~/
```
